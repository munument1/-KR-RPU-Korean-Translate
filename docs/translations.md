## 다국어 번역 안내 (Translations)

- 대부분의 번역은 **미완성** 상태이며 있는 그대로 제공됩니다. 미완성된 번역을 사용하는 경우 게임 플레이 도중에 일부 영문 대사나 텍스트가 표시될 수 있습니다.
- [번역 기여 웹페이지](https://hive.bgforge.net/projects/fallout/rp/)에서 번역 완성 작업을 도울 수 있습니다. 세부적인 기여 방식은 [포럼](https://forums.bgforge.net/viewtopic.php?f=9&t=22)을 참조해 주세요.

### 한국어 번역 리소스 적용 안내
- 본 저장소의 한국어 패치는 한국어 번역을 직접 설치할 수 있도록 구조화되어 있습니다.
- 수동 설치를 원하는 경우:
  1. 본 저장소의 `mods` 및 `appearance` 폴더의 내용을 복사하여 폴아웃 2 설치 폴더에 덮어씌워 줍니다.
  2. `fallout2.cfg` 파일 내의 `language=english` 부분을 사용하려는 언어 디렉토리에 맞게 맞춰 줍니다. (예: 번역 파일이 `mods/fo1_base/text/english`나 `mods/rpu/text/english` 등의 영문 디렉토리 구조를 직접 덮어쓰도록 설계된 경우에는 기본값인 `language=english` 설정을 그대로 유지해야 한글이 출력될 수 있습니다.)
