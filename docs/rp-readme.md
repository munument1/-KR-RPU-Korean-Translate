# Killap의 폴아웃 2 복원 프로젝트 - 버전 2.3.3 (설치 정보)

## 개요
이 프로젝트의 목적은 오리지널 폴아웃 2 게임에서 예산, 시간 등의 문제로 인해 개발이 중단되거나 더미 데이터로만 남아 있던 모든 콘텐츠(지역, NPC, 퀘스트, 아이템 등)를 게임 내에 정상적으로 구현하고 복원하는 것입니다. 이 모드는 폴아웃 특유의 고유한 분위기를 해치지 않고 원작 개발진이 원래 의도했던 기획을 최대한 충실하게 살려내고자 노력했습니다. 일부 사안에서는 제작자의 판단이 개입되기도 하였으나, 핵심 요소들은 모두 원래 블랙아일 폴아웃 2 개발진이 구상했던 내용입니다.

---

## 중요 안내 사항
* **오래된 세이브 파일 호환 불가:** 본 모드를 설치한 후에는 기존의 세이브 파일을 사용할 수 없으며, 반드시 **새 게임**을 시작해야 합니다.
* **설치 성공 확인법:** 
  1. 게임 시작 직후 시련의 사원(Temple of Trials) 바로 외부의 클린트(Klint) 뒤편을 보면 작은 바위가 하나 있습니다. 이 바위를 '조사(Look)'하면 모드의 상세 버전이 텍스트로 표시됩니다.
  2. 또한 게임 메인 메뉴 화면 우측 하단에 복원 프로젝트의 버전 번호가 표시됩니다. 
  3. 이 두 가지(사원 앞 바위 텍스트와 메인 메뉴 버전 표시)가 모두 확인되어야 모드가 100% 정상 설치된 것입니다.

## 자주 묻는 질문 (FAQ)

### Q: 공식 1.02 패치나 별도의 비공식 패치(UP)를 먼저 설치해야 하나요?
**A:** 아닙니다. 공식 패치 및 비공식 패치(Unofficial Patch)의 모든 수정 사항이 본 복원 프로젝트 모드 패키지에 이미 통합되어 포함되어 있습니다.

### Q: 오리지널 순정(Vanilla) 버전의 세이브 파일을 불러올 수 있나요?
**A:** 불가능합니다. 이전의 세이브 파일들은 구조적 차이로 인해 전혀 호환되지 않습니다. 다만, 향후 복원 프로젝트 버전이 새로 업데이트되는 경우에는 특별한 공지가 없는 한 본 모드의 이전 버전 세이브 파일과 호환됩니다.

### Q: 아동 삭제 버전(유럽 판본 등)을 쓰고 있습니다. 아이들을 복원하려면 따로 패치를 깔아야 하나요?
**A:** 아닙니다. 이 모드에 포함된 비공식 패치가 게임 내에 누락된 아동 이미지 및 그래픽 리소스를 자동으로 복원해 줍니다.

### Q: 게임 실행 시 'Failure initializing input devices' 오류와 함께 튕깁니다.
**A:** 폴아웃 2 폴더에 있는 `ddraw.ini` 파일을 엽니다. `[input]` 섹션에서 `BackgroundKeyboard` 값을 `1`로 수정하고 실행해 보세요. 그래도 해결되지 않으면 `BackgroundMouse` 값도 `1`로 수정해 보시기 바랍니다.

### Q: 화면이 검은색으로 나오거나 색상이 깨져서 출력됩니다 (사이케델릭 현상).
**A:** 폴아웃 2 폴더의 `ddraw.ini` 파일을 열고, `[Graphics]` 섹션에서 `Mode` 값을 `4`로 변경해 줍니다.

### Q: 실행 시 'Fallout appears to be running in compatibility mode' 메시지가 뜨며 실행이 되지 않습니다.
**A:** 윈도우 호환성 모드가 켜져 있어서 발생하는 문제입니다. `fallout2.exe` 파일을 우클릭하여 속성 창으로 간 뒤, 호환성 탭에서 모든 체크 항목을 해제해 주세요.

### Q: 월드맵 이동 속도가 너무 빠르거나 느립니다. 어떻게 조절하나요?
**A:** 원래는 CPU의 클럭 속도에 비례해 속도가 빨라지는 엔진 한계가 있었으나, 본 모드는 1990년대 말 구동 환경에 맞춰 속도를 보정해 줍니다. 그럼에도 조절을 원하신다면 폴아웃 2 폴더 내의 `ddraw.ini` 파일을 열고 `WorldMapDelay2=` 항목을 찾으세요. 기본값은 `WorldMapDelay2=66` 이며, 이 값을 높이면 속도가 느려지고 값을 낮추면 월드맵 이동 속도가 빨라집니다.

### Q: 게임 해상도는 어떻게 변경하나요?
**A:** Mash의 고해상도 패치 덕분에 640x480 이상의 고해상도를 지원합니다. 인게임 메인 메뉴의 **설정(Options)** -> **화면 설정(Screen Settings)**에서 원하는 해상도로 구성한 뒤 완료를 누르시면 저장됩니다. 또는 게임 폴더에 있는 `f2_res_Config.exe`를 실행하여 게임 밖에서 구성할 수도 있습니다.

### Q: 플레이할 때 CPU 점유율이 100%까지 치솟습니다.
**A:** 게임 폴더 내의 `ddraw.ini` 파일 중 `[Misc]` 섹션에서 `ProcessorIdle` 설정을 `1`로 변경해 줍니다. 그래도 해결되지 않으면 해당 옵션을 원래대로 `-1`로 돌려둔 뒤, `f2_res.ini` 파일을 열어 `CPU_USAGE_FIX` 설정을 `1`로 지정해 주세요. (두 옵션을 동시에 1로 설정하지는 마세요.)
