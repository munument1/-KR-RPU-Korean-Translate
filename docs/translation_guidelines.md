# BGforge 번역 시작 가이드 (How to start translating)

이 가이드는 BGforge의 공식 번역 가이드 포럼 게시물([How to start translating](https://forums.bgforge.net/viewtopic.php?t=25))을 기반으로 번역한 지침서입니다.

---

## 1. 번역 시스템 이해하기
BGforge는 Weblate 번역 시스템을 사용하여 번역을 진행합니다. 번역 참여 전에 아래의 링크(영문)들을 참조하여 시스템 구성을 이해해 주세요:
- [제안 및 수정 사항 (Suggestions and revisions)](https://forums.bgforge.net/viewtopic.php?f=9&t=50&p=136)
- [번역 용어집 (Glossary)](https://forums.bgforge.net/viewtopic.php?f=9&t=49)
- [여성 어투 번역 규칙 (Female translation)](https://forums.bgforge.net/viewtopic.php?f=9&t=28)

---

## 2. 가입 및 설정
1. [계정 등록 페이지](https://tra.bgforge.net/accounts/register/)에서 계정을 만듭니다.
2. 프로필 설정에서 기여하고자 하는 언어(Korean 등)를 선택합니다. (자세한 설정 방법은 [Weblate 공식 문서](https://docs.weblate.org/en/latest/user/profile.html) 참조)

---

## 3. 번역 지침 및 팁 (Tips)

### 번역 방법
- **모국어(한국어) 사용자:** 다른 참여자가 올린 **"제안(suggestions)"**을 확인하는 것부터 시작하는 것이 좋습니다. 제안 내용이 자연스럽고 양호하다면 승인하세요. 수정이 필요하다면 **"수락 후 편집(Accept and Edit)"**을 하거나, 삭제 후 처음부터 다시 번역해 주시기 바랍니다.
- **비모국어 사용자:** **"검토 대상(review)"**으로 표시된 문장부터 시작하는 것이 쉽습니다. 번역에 확신이 선다면 저장하고, 그렇지 않다면 모국어 사용자가 나중에 검토할 수 있도록 **"제안(suggestion)"** 상태로 남겨두세요.
- 번역 작업 속도를 높이기 위해 다양한 [키보드 단축키](https://docs.weblate.org/en/latest/user/translating.html#keyboard-shortcuts)를 적극 활용해 보세요.

### ⚠️ 기계 번역(번역기) 사용 시 유의사항 (Machine translation note)
- 제공되는 "기계 번역" 기능은 번역가의 작업을 돕기 위한 **참고용 데이터**일 뿐이며, 번역가를 대체할 수 없습니다. **번역기 결과를 무작정 복사하여 붙여넣지 마십시오.**
- 만약 본인이 직접 매끄럽게 번역하기 어려운 까다로운 문장을 마주한다면, 직접 억지로 번역하지 말고 그냥 건너뛰어 다른 능력 있는 번역가가 작업할 수 있도록 남겨두세요. **품질이 낮은 번역 결과를 양산하는 것은 BGforge 프로젝트의 목적이 아닙니다.**
