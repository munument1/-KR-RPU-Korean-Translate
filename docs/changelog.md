## 변경 이력 (Changelog)

- 이 파일은 RP 2.3.3 이후의 복원 프로젝트(RPU) 변경 내역만 기록합니다.
- 아래 목록 외에도, RPU 릴리즈에는 대응하는 [UPU 릴리즈(Unofficial Patch)](https://github.com/BGforgeNet/Fallout2_Unofficial_Patch/blob/master/docs/changelog.md)의 모든 버그 수정 사항이 포함되어 있습니다.
- UPU와 RPU는 종종 독립적으로 업데이트될 수 있으므로 버전 번호에 건너뜀이 있을 수 있습니다.
- 버전 `2.4.*` 제품군은 Pixote가 제작한 [업데이트된 맵](https://forums.bgforge.net/viewtopic.php?t=371)과 관련 스크립트 수정 사항들을 포함합니다. 그 외에는 동일한 패치 레벨 규칙을 따릅니다.

*(상세 변경 이력은 원본 기술 문서 내용 보존을 위해 영문으로 기록되어 있습니다.)*
